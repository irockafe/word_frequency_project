Create Russian Word-Frequency Lists

Study vocabulary based on how often it's used, in texts you want to read.

Upload a .txt file

Currently, only plain-text (.txt) files are supported.

While the program should be able to create frequency lists for any language where words are separated by spaces (sorry, Chinese!), it can only lemmatize words in Russian thanks to pymorphy2.

In non-linguist terms, that means that walk, walks, and walked will be counted as separate words. But хожу, ходит, и ходили will all be counted as ходить.

100 pages of text can be processed in roughly 90 seconds.
Hopefully more languages, file formats, and features will come in the future.
